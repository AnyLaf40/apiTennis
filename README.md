# apiTennis

__L'objectif principal de cette API est de repertorier les matchs de tennis en France.__

__Vous aurez donc accès, via des requêtes HTTP, à des données telles que les joueurs, les rencontres, les résultats, les stats,...__

__Tout d'abord, voici une procédure d'installation afin de manipuler l'API sur un serveur local.__

## Procédures d'installation

### Wampserver

__Commencez par installer WAMPSERVER.__

### PhpMyAdmin

__Ensuite, lorsque WAMPSERVER est installé, accédez à [phpmyadmin](http://127.0.0.1/phpmyadmin/) et créez un compte sous MySQL.__
__Après cela, connectez-vous avec votre compte et importez le fichier "base.sql" présent sous le dossier "apiTennis/SQL" du projet Github.__
